# 米窗3-全局小窗

[English](https://www.github.com/sunshine0523/Mi-Freeform/README.md)

米窗3是一款Android第三方小窗软件，支持Android 8.1-Android 13。

## 缺陷

- 触控采样率
- 还原到默认屏幕后部分应用会重载
